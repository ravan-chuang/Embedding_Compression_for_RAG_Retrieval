## AI Embedding Compression for Efficient Semantic Search

**Tech Stack:** Python, NumPy, scikit-learn, sentence-transformers, AG News  
**Field:** AI Systems, Semantic Search, Data Compression, Vector Retrieval

### Summary

Built a benchmark pipeline to evaluate embedding compression methods for semantic search systems. Compared scalar quantization, vector quantization, product quantization, and random rotation using compression ratio and retrieval quality metrics.

### Highlight

Random Rotation + int4 achieved:

- **8.00x compression**
- **0.9780 average cosine similarity**
- **0.9058 Top-5 Recall**

### Why It Matters

This project addresses a real AI infrastructure issue: reducing embedding storage cost while maintaining retrieval quality for RAG, semantic search, and vector database systems.
